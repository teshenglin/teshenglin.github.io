Multidimensional scaling

Te-Sheng Lin (林得勝)
Last updated on Sep 7, 2020 3 min read Calculus, Computational Mathematics, Linear algebra

Multidimensional scaling, 簡稱 MDS, 是個資料分析或是資料降維的工具.

這裡我們要談一下從數學角度來說 MDS 的原理及做法.

假設我們有 $n$ 筆 $p$ 維的資料, 記成 $$ \{x_1, x_2, \cdots, x_n\} \in R^p $$ 那我們可以據此構造出一個距離平方矩陣 $D$, Euclidian distance matrix, 簡稱 EDM, 其中 $D_{ij} = \|x_i-x_j\|^2$.

而 MDS 要做的事是反過來, 假設我們拿到一個 EDM 矩陣, 我想要找出原始的點 $x_i$ 使得他所構成的 EDM 就是我們初始拿到的這個. 這樣子的問題稱為 classical MDS.

廣義一點的 MDS, 其一可以想像是, 我們拿到的距離也許不是用歐式距離量的, 也許是某個 weighted inner product space 裡的距離. 其二就是, 我們拿到的距離矩陣也許其原始生成的資料點在很高維的空間裡, 那我是否能找到一組低維度空間中的點, 使得這組點所生成的距離矩陣跟我拿到的距離矩陣很近?

我們先定義一個矩陣稱為置中矩陣 $H$, centering matrix, 也就是把一組資料平移使得其中心為坐標原點: $$ H = I_n - \frac{1}{n}\bf{1}\bf{1}^T, $$ 其中 $I_n$ 是 $n\times n$ 的單位矩陣, $\bf{1}$ 則是元素全為 $1$ 的 $n\times 1$ 向量.

若我們將原始資料收集一起成一個 $p\times n$ 矩陣 $X = [x_1, \cdots, x_n]$, 則有 $$ X H = X (I_n - \frac{1}{n}\bf{1}\bf{1}^T) = X - \frac{1}{n}\left(\sum_i x_i\right) \bf{1}^T = X - \mu \bf{1}^T = Y, $$ 其中 $\mu=\frac{1}{n}\sum_i x_i$ 就是原始資料的平均, $Y=[y_1, \cdots, y_n]$, $y_i = x_i-\mu$. 所以的確 $XH$ 就是把資料平移使得其中心為坐標原點.

推導一下後也可以發現 $H\hat{X}$ 則是將 $\hat{X}$ row space 的資料平移到原點.

我們有以下這個定理

Theorem

給定原始資料 $X = [x_1, \cdots, x_n]$, 且其相對應的 EDM 為 $D$, 則我們有 $$Y^T Y = -\frac{1}{2}HDH.$$

這個定理告訴我們的是, 如果我們只知道一組資料的 EDM 矩陣 $D$, 那要怎樣把資料給還原回來.

Remark: 當然, 不可能把原始資料完全還原回來! 如同定理中所述我們所能算出的等號左邊是 $Y$, 也就是置中後的原始資料. 事實上我們也很容易想像, 若將一組資料平移或是旋轉後其 EDM 應該是完全不會變的. 也就是說我們若只知道 EDM, 則其原始資料應該有無限多解. 這裡所謂的還原回來是找到其中一組解, 其他所有可能的解則都可以將之平移或旋轉後得到.

定理等號右邊由於是個對稱矩陣, 所以可以對角化為 $$ -\frac{1}{2}HDH = V\Lambda V^T = V\sqrt{\Lambda}\sqrt{\Lambda} V^T, $$ 其中 $\Lambda$ 是個對角矩陣包含所有特徵值, 而 $\sqrt{\Lambda}$ 則是將 $\Lambda$ 對角線元素都開根號.

跟定理對照一下可以輕易地看出來, 平移後的原始資料點可以被還原出來: $Y = \sqrt{\Lambda}V^T$.

Remark: 若原始資料 $x_i\in R^p$, 則 $Y^TY$ 的 rank 為 $p$, 必有 $n-p$ 個為零的特徵值. 所以, 若我們將 $-\frac{1}{2}HDH$ 對角化後發現有 $m$ 個為零的特徵值, 表示原資料的 $p=n-m$, 那我們就把這些零特徵值都拿掉, 使 $\sqrt{\Lambda}$ 為一個 $p\times n$ 的矩陣, 這樣我們就有 $Y = \sqrt{\Lambda}V^T \in R^{p\times n}$.

給定一個 $n\times n$ 的 EDM 以及某正整數 $k$, MDS 簡單的程式如下:

% Matlab program
function Y = multidimensional_scaling(D, k)
    n = size(D, 1);                         % n 個 samples
    mu = sum(D, 1)/n; D = D - mu;           %
    mu = sum(D, 2)/n; B = D - mu;           %
    B = -0.5*B;                             % B = -0.5*H*D*H
    [V, D] = eig(B, 'vector');              % 求出特徵值及特徵向量
    [sqD, ind] = sort(sqrt(D), 'descend');  % 將特徵值按大小排列
    sqD = sqD(1:k);                         % 取前 k 個
    V = V(:, ind);                          % 將特徵向量照樣排列
    Y = V(:,1:k)'.*(sqD*ones(1,n));         % 求出 k 維資料點
end
